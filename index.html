SQL Database Acceleration in Hardware

About

The era of big data is upon us, and the footsoldiers of this new generation are ill equipped and ill suited for the task. General purpose processors are inefficient and slow, while GPUs are power hungry and better suited for graphics-like vector processing.

Enter dau. dau stands for Database Acceleration Unit. It is a partial implementation of the SQL language in hardware, with the end goal of FPGA-based database acceleration. Working in parallel with, and being arbitrated by existing general purpose CPUs, dau provides a faster, cheaper, and more power efficient way to process data.
